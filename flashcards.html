Flashcards

I like flashcards. A lot. I generate them using a program I wrote which I'll be releasing shortly. I like printing directly onto index cards.

I update this page regularly, including revising files already present. If you want to keep abreast of changes you should watch my repository on GitHub.

I will be adding an 8.5" x 11" output mode for printing on standard paper or the expensive perforated kind.

I welcome feedback on the visual appearance of the pdfs, or on the flashup markup language. I also appreciate notifications of typos or mistakes in the cards themselves.
